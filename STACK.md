# STACK.md

## Chosen stack

| Layer | Choice | Why |
|---|---|---|
| Generator | Astro 4 | Zero-JS by default, component reuse, first-class Cloudflare Pages support |
| Language | TypeScript (strict) | Catches data-shape errors in project entries at build time |
| Styling | Plain CSS + custom properties | No build complexity, no framework, full control |
| Font | JetBrains Mono (Google Fonts) | Single mono family, two weights, free, fast |
| Hosting | Cloudflare Pages | Free, fast global edge, no card required |
| Version control | Git + GitHub | Source of truth |

## Dependencies — exhaustive list

```json
{
  "dependencies": {
    "astro": "^4.16.0"
  }
}
```

That's it. No other runtime dependencies. No dev dependencies needed beyond what Astro pulls in.

If you find yourself wanting to add another package, stop and ask why.

## Build output target

- `npm run build` produces `dist/`
- Total `dist/` size: < 100KB
- Per-page HTML: < 15KB uncompressed
- No JS files in `dist/` except auto-generated Astro internals (which should also be near-zero)
